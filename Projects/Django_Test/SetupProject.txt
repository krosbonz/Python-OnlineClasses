Start by creating a folder for your project. In this case it is;
Python\Projects\Django_Test

Setup virtual environment
pip install virtualenv (only need to do this once)
Create env folder (django_test_env)
CD into that folder and create a virtual environment;
virtualenv nameofenvironment (django_test_env)
Activate virtual env by running: \nameofenvironment\Scripts\activate
Your PowerShell prompt will change to:
(django_test_env) PS C:\Users\hauck\...
Install django
Run: pip install django
If not there CD to project folder
Run: django-admin startproject nameofdjangoproject


To run the Django server:
cd django_test_proj
python manage.py runserver
